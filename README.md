# 🚌 Campus Ride - Smart College Bus Tracking System

Campus Ride is a smart web-based college bus tracking system that enables students to track buses in real time while allowing drivers and administrators to manage routes, schedules, and fleet operations efficiently.

> ⚠️ **Note:** This project was developed with the assistance of AI-powered development tools. The project requirements, features, customization, integration, and testing were carried out by me.

---

## ✨ Features

- 🚌 Real-time bus tracking
- 📍 Live GPS location on interactive maps
- 👨‍🎓 Student Dashboard
- 👨‍✈️ Driver Dashboard
- 👨‍💼 Admin Dashboard
- 📅 Bus schedule management
- 🛣️ Route management
- ⏱️ Estimated Time of Arrival (ETA)
- 🔔 Real-time notifications
- 📊 Analytics Dashboard
- 🔐 Firebase Authentication
- ☁️ Firebase Realtime Database integration
- 📱 Responsive UI
